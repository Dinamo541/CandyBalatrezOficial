# Candy Balatrez

Candy Balatrez es un juego de rompecabezas inspirado en el clásico estilo de "match-3". El objetivo principal es completar niveles al combinar gemas del mismo tipo para alcanzar un puntaje objetivo antes de que se agoten los movimientos disponibles.

## Características principales

- **Niveles progresivos**: Tres niveles con diferentes configuraciones y objetivos.
- **Animaciones dinámicas**: Gemas que caen, desaparecen y se intercambian con animaciones suaves.
- **Elementos especiales**: Bombas y bloques que añaden complejidad al juego.
- **Interfaz gráfica**: Diseñada con SFML para una experiencia visual atractiva.

## Requisitos del sistema

- **Sistema operativo**: Windows
- **Dependencias**: 
  - [SFML](https://www.sfml-dev.org/) (Simple and Fast Multimedia Library)
  - C++ compilador compatible con C++11 o superior
